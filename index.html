Programming

Programming adalah sebuah proses untuk membuat program di komputer. Program yang dibuat bisa berupa software, website, aplikasi android, dsb.
Jika kalian memilih programming, kalian akan diajak untuk mempelajari tentang berbagai macam bahasa pemogramman.
Kalian akan belajar bagaimana caranya membuat sebuah website dari awal menggunakan bahasa pemogramman seperti HTML,CSS, dan Javascript.
Web design juga termasuk didalamnya agar website yang kalian buat nantinya bisa terlihat lebih menarik.

Networking

Dalam bidang ini kalian akan diajak untuk mengeskplorasi lebih dalam mengenai jaringan komputer.
Cara menghubungkan komputer ke dalam sebuah jaringan dengan berbagai perantara seperti swtich, router dsb.
Dari sini kalian juga akan bisa memilih untuk mempelajari tentang dunia cyber security.
